# Project Limitations & Disclaimer

This project is a geospatial data-engineering demonstration designed to execute a reproducible tree-cover loss spatial analysis. It is explicitly **not** a formal carbon MRV (Monitoring, Reporting, and Verification) tool, nor does it make any claims suitable for carbon credit certification.

## 1. Tree-Cover Loss ≠ Deforestation
The Hansen Global Forest Change dataset maps stand-replacement disturbances (tree-canopy closure dropping from >25% to below that threshold). 
A flagged pixel is a screening signal for **tree-cover loss**, which can be caused by:
- Legal logging or timber harvesting
- Natural disasters (fires, droughts, windstorms)
- Agricultural crop conversion or shifting cultivation
- Plantation rotation

It does not inherently prove permanent, anthropogenic **deforestation**, nor does it investigate legality or land tenure.

## 2. No Carbon or Biomass Calculation
This tool performs a purely 2D spatial area calculation (hectares). It does not:
- Model above-ground biomass (AGB)
- Calculate carbon stock
- Compute CO₂e emissions
- Assess additionality or baseline emission scenarios

## 3. Illustrative Area of Interest (AOI)
The GeoJSON AOI provided (near Prey Lang, Cambodia) is an illustrative bounding box used strictly to demonstrate the pipeline on a region with known historical canopy disturbance. It is not a legally surveyed project boundary, and no claims should be made regarding the actual jurisdiction or conservation projects operating in that region.

## 4. No Certification Authority
The results generated by this pipeline cannot be used for compliance, regulatory approval, or voluntary carbon market certification (e.g., Verra, Gold Standard, Equitable Earth). Proper certification requires independent, third-party, ground-truthed auditing.
